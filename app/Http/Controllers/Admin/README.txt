#   Puesto que generalmente ambos pueden fungir como instructores, sin embargo, hay algunas acciones
#   que sólo el administrador puede realizar y deben ser validadas previamente a su ejecución.
# 
#   Cada controlador cuenta con las funciones de "resource" (que genera rutas para cada uno de los 
#   métodos), los cuales dejo de manera explícita para su futuro uso si es necesario.
